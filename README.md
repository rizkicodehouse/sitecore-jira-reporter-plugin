# JIRA Reporter Plugin

Sitecore Marketplace Pages Context Panel plugin for filing JIRA bugs.

## Version pinning

`@sitecore-marketplace-sdk/*` is pinned to an exact 0.3.x version
because the design was built against the 0.3 API (pages-context
subscriptions, SDK v0.3 permissions). Upgrade deliberately after
re-running the research in `JIRA/CLD-XXX-JIRA-Module-Reporter/research/`.
